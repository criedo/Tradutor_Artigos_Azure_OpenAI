# Tradutor_Artigos_Azure_OpenAI
Atividade para o bootcamp Microsoft Certification Challenge #5 - AI 102 que desenvolve uma solução de tradução automática de artigos técnicos utilizando o AzureAI, com foco em garantir precisão terminológica e contexto específico do domínio técnico, facilitando o acesso a conteúdos especializados em diferentes idiomas.
